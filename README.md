# MERN E-Commerce Platform

A full-stack E-Commerce web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This project provides a modern online shopping experience with user authentication, product management, cart functionality, order processing, and an admin dashboard.

## Features

### User Features

* User Registration & Login
* Secure Authentication
* Browse Products
* Product Search & Filtering
* Shopping Cart
* Order Placement
* Order History
* Responsive Design

### Admin Features

* Admin Dashboard
* Product Management (Add, Update, Delete)
* User Management
* Order Management
* Inventory Control

## Tech Stack

### Frontend

* React.js
* React Router
* Axios
* CSS / Tailwind CSS

### Backend

* Node.js
* Express.js
* JWT Authentication
* REST API

### Database

* MongoDB
* Mongoose

Run Both Servers

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```


### Run Backend

```bash
npm start
```

### Run Frontend

```bash
npm run dev
```
